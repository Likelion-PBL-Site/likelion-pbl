---
description: 세션 진행 상황 저장
---

현재 세션의 진행 상황을 `.claude/session-progress.md` 파일에 저장합니다.
이 파일은 세션이 끊겼을 때 이어서 작업할 수 있도록 컨텍스트를 보존합니다.

## 저장 절차

1. `.claude/session-progress.md` 파일을 읽어서 기존 내용 확인 (없으면 새로 생성)

2. 현재 대화에서 다음 정보를 추출하여 저장:

```markdown
# 세션 진행 상황

> 마지막 업데이트: YYYY-MM-DD HH:MM

## 현재 작업 중인 태스크
[현재 진행 중인 작업 설명]

## 진행 상황
- [x] 완료된 단계
- [ ] 진행 중인 단계
- [ ] 남은 단계

## 핵심 컨텍스트
- 문제/목표: [간단한 설명]
- 선택한 접근 방식: [방식 설명]
- 관련 파일: [파일 경로들]

## 다음 단계
[세션 재개 시 해야 할 일]

## 대화 요약
[중요한 의사결정 포인트 요약]
```

3. 저장 완료 후 사용자에게 알림:
   - 저장된 파일 경로
   - 핵심 저장 내용 요약 (2-3줄)

## 세션 재개 시

새 세션 시작 시 `.claude/session-progress.md` 파일이 있으면:
1. 자동으로 읽어서 이전 컨텍스트 파악
2. 사용자에게 "이전 작업을 이어서 할까요?" 제안

## 자동 저장 트리거

다음 상황에서는 자동으로 체크포인트 저장을 제안하거나 수행:
- 큰 작업 단위 완료 시
- 의사결정 포인트 도달 시
- 여러 파일 수정 완료 시
- 사용자가 "잠깐" 또는 "나중에" 언급 시
